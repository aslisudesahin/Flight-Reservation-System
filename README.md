# Flight-Reservation-System
A comprehensive Flight Reservation System developed with C# and .NET. Features distinct Admin and Customer interfaces, real-time flight scheduling, secure user authentication, and full SQL Server integration for robust data management.
Flight Reservation System
This is a robust desktop application for managing airline reservations, built using the C# .NET Framework and SQL Server.

 Core Features
Dual User Roles: Specialized dashboards for Administrators (to manage flights) and Customers (to book tickets).

Flight Management: Admins can add, update, or delete flight schedules and aircraft information.

Booking Engine: Real-time flight searching and seat reservation system.

Data Persistence: Uses a structured SQL database to store users, flights, and reservation records securely.

User Profiles: Functionality for new user registration and existing user login.

 Technical Details
Language: C#

UI Framework: Windows Forms (WinForms)

Database: SQL Server (T-SQL)

Architecture: Modular design with separate classes for Models (Flight, User, Reservation) and DataAccess.
